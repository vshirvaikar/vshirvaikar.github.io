BLOG POST WORKFLOW (NOTEBOOK -> MARKDOWN -> HTML)

1) Run the notebook and export figures
- Run all cells to generate model outputs and figures.
- Export figures to `../visuals`, and keep all output paths relative to the post folder.
- Save each figure you want to export as a named matplotlib object, for example `fig_name = fig`.
- Keep a reusable export helper in the notebook that sets figure size from `width_px`, `aspect`, and `dpi`, 
  optionally resets font sizes, saves to `../visuals`, and prints a markdown `<img ... />` snippet for the exported file.
- For all picture formatting and export conventions, use these notebooks as the reference implementation:
  `blog/2026.03 National Teams/analysis/model.ipynb`
  `blog/2026.04 CFB Preseason/analysis/model.ipynb`

2) Write markdown article
- Add YAML frontmatter for title/subtitle:
  ---
  title: "..."
  subtitle: "..."
  ---
- For images: use <img class="post-figure" src="./visuals/file.png" />
- For complex tables: use HTML with class-based CSS for layout control.
- For equations: use display math ($$...$$) for proper LaTeX rendering.
- The back-link and site chrome are injected by blog-template.html; no back-link needed in the MD.
- Figure and table styling are centralized in blog-template.html; only add a CSS block in MD for true one-off overrides.

3) Compile to single-file HTML
  cd blog/<post-folder>
  pandoc <post>.md \
    -o <post>.html \
    --standalone \
    --self-contained \
    --resource-path=".:visuals" \
    --template=../blog-template.html \
    --mathjax

4) Verify output
- Open HTML in browser.
- Check: images load, tables render, equations display, no warnings.

5) Update RSS feed
- Add a new <item> block to blog/feed.xml with:
  <title>, <link>, <pubDate> (RFC 822 format), <guid>, and <description> (excerpt)
- Update <lastBuildDate> in the <channel> to today's date.
- Use real page URLs only in RSS (no fragment links like #blog in <channel><link> or <item><link>). 
- Feed readers will auto-discover via the <link rel="alternate"> in the template.
